Отправка токенов

  • Выбор токена: переключение между SOL (0.00 доступно) и TNG (1000 доступно) с real-time балансами.
  • Гибкий получатель: вкладки «Адрес» и «@Username» для отправки по Solana адресу или поиску пользователей.
  • Анонимные переводы: чекбокс «Отправить анонимно» скрывает отправителя в истории получателя.
  • Memo поле: необязательный комментарий к транзакции для описания цели перевода.
  • QR сканер: кнопка справа от поля адреса для автозаполнения через камеру.
Smart Contract Integration
Pre-validation: Проверка баланса, формата адреса и лимитов перед отправкой транзакции в блокчейн.
Username Resolution: API поиск пользователей по username с проверкой регистрации и получением wallet адреса.
